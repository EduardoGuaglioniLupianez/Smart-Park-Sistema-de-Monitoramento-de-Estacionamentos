# [cite_start]Smart-Park: Sistema de Monitoramento de Estacionamentos [cite: 58, 72]

[cite_start]O **Smart-Park** é uma solução de IoT desenvolvida para a gestão e monitoramento inteligente de vagas em estacionamentos, visando otimizar o fluxo de veículos e melhorar a experiência do usuário[cite: 58, 72].

## 📂 Estrutura do Projeto

Abaixo estão detalhados os diretórios e arquivos que compõem este repositório:

### [cite_start]1. 🗄️ Banco de Dados (`/Banco de Dados`) [cite: 76]
[cite_start]Contém os arquivos necessários para a configuração da infraestrutura de dados do projeto[cite: 58, 76].
* [cite_start]**`SCRIPT_BD_SPRINT1.sql`**: Script SQL contendo a criação das tabelas e a estrutura inicial do banco de dados para a primeira fase do projeto[cite: 58, 59].

### [cite_start]2. 🌐 Códigos e Site (`/Códigos e Site`) [cite: 76]
[cite_start]Nesta pasta reside a interface visual e a lógica de simulação financeira para o cliente final[cite: 63, 67, 76].
* [cite_start]**`TelaSimuladorFinanceiro.html`**: Página web que apresenta a interface do sistema, permitindo interações do usuário e visualização de dados[cite: 67, 68].


### [cite_start]3. 🤖 Código Arduino (`/Código_Arduino_Teste`) [cite: 71, 76]
[cite_start]Contém o software embarcado que controla os sensores e a comunicação do hardware[cite: 71, 76].
* [cite_start]**`codigo_arduino.ino`**: Código-fonte desenvolvido para a placa Arduino, responsável por realizar a leitura dos sensores de presença nas vagas e enviar os dados para o sistema[cite: 71].

---

## 🛠️ Tecnologias Utilizadas
* **Hardware:** Arduino e Sensores IoT[cite: 71].
* [cite_start]**Banco de Dados:** SQL (Estrutura de tabelas)[cite: 58, 60].
* [cite_start]**Frontend:** HTML5, CSS e JavaScript[cite: 67, 70].

## 📝 Como utilizar
1.  [cite_start]**Banco de Dados:** Execute o script `.sql` em seu gerenciador de banco de dados para preparar as tabelas necessárias[cite: 58, 60].
2.  [cite_start]**Hardware:** Carregue o arquivo `.ino` na sua placa utilizando a Arduino IDE para iniciar o monitoramento das vagas[cite: 71].
3.  [cite_start]**Interface:** Abra o arquivo `.html` em um navegador para acessar o simulador financeiro e a interface do usuário[cite: 67].
